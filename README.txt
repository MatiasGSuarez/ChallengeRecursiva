El proyecto lo realicé con .net7 y blazor. Utilicé syncfusion y bootstrap para los componentes visuales. 
Al clonar el proyecto y ejecutarlo en el SIDEBAR se va a poder ver la etiqueta LISTA DE SOCIOS y la etiqueta ESTADISTICAS DE EQUIPOS, en esos componentes se cumplen las consignas requeridas.
No se va a poder navegar desde "Lista de Socios" directamente a "Estadisticas de Equipos" ya que no es una versión paga de Syncfusion por lo que deberá reiniciarse la ejecución del proyecto.
